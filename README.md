# Jwong1mhs Catalina-legacy

## How do I install these formulae?

`brew install jwong1mhs/catalina-legacy/<formula>`

Or `brew tap jwong1mhs/catalina-legacy` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jwong1mhs/catalina-legacy"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
